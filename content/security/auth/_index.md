# Authentication and access control

Union.ai provides a layered identity and access management system that controls how users and applications authenticate, what resources they can access, and how tenant isolation is enforced.

This section covers:

* [Authentication methods](./authentication): OIDC, API keys, and service accounts.
* [SSO configuration](./sso-configuration): Single sign-on setup and security properties.
* [Role-based access control](./rbac): Built-in roles, custom policies, and the least-privilege principle.
* [Organization and tenant isolation](./organization-isolation): Database-layer, data plane, and service-level isolation.
* [Human access controls](./human-access-controls): How Union.ai personnel access customer environments in Self-Managed and BYOC deployments.
