# AttrDict

**Package:** `flytekit.extras.tasks.shell`

Convert a dictionary to an attribute style lookup. Do not use this in regular places, this is used for
namespacing inputs and outputs



```python
class AttrDict(
    args,
    kwargs,
)
```
| Parameter | Type | Description |
|-|-|-|
| `args` | `*args` | |
| `kwargs` | `**kwargs` | |
