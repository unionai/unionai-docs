# Data protection and encryption

Union.ai protects customer data through cloud-native encryption, a strict data classification framework, and strong data residency guarantees. All customer data is encrypted both at rest and in transit using industry-standard protocols.

This section covers:

* [Encryption at rest](./encryption-at-rest): Encryption standards and key management for all storage types.
* [Encryption in transit](./encryption-in-transit): TLS, mTLS, and HTTPS enforcement across all communication paths.
* [Data classification and residency](./data-classification-and-residency): How data is classified, where it resides, and multi-cloud region support.
